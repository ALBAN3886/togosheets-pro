Modules de formation

6 MODULES COMPLETS

Du HTML de base jusqu'au déploiement en ligne, chaque module construit sur le précédent pour une progression naturelle.

01
🌐
HTML5 — Structure Web

Apprenez à structurer le contenu de vos pages web avec les bonnes balises sémantiques.

  • Structure d'une page HTML (head, body, sections)
  • Balises sémantiques : header, nav, main, footer
  • Formulaires HTML complets
  • Tableaux, listes, médias (images, vidéos)
  • Liens et ancres de navigation
  • Bonnes pratiques SEO de base
⏱ Durée : 1 semaine — Niveau : Débutant
02
🎨
CSS3 & Design

Stylisez, animez et rendez votre site responsive pour tous les écrans.

  • Sélecteurs, propriétés, box model
  • Flexbox & CSS Grid (mises en page modernes)
  • Design responsive (mobile-first)
  • Animations et transitions CSS
  • Variables CSS (custom properties)
  • Typographie, couleurs, dégradés
⏱ Durée : 1 semaine — Niveau : Débutant
03
⚡
JavaScript

Ajoutez l'interactivité, la logique et les appels API à vos pages web.

  • Variables, fonctions, conditions, boucles
  • Manipulation du DOM (document HTML)
  • Événements utilisateur (clic, saisie...)
  • Fetch API & appels serveur
  • JSON et traitement de données
  • Stockage local (localStorage)
⏱ Durée : 2 semaines — Niveau : Intermédiaire
04
🔥
Firebase / Base de données

Lisez, écrivez et synchronisez vos données en temps réel avec Firebase.

  • Introduction à Firebase & Firestore
  • Authentification utilisateurs (email/mdp)
  • Lecture et écriture de données en temps réel
  • Règles de sécurité Firebase
  • Firebase Storage (images, fichiers)
  • Projet pratique : app connectée complète
⏱ Durée : 2 semaines — Niveau : Intermédiaire
05
🐙
GitHub

Versionnez votre code, collaborez sur vos projets et ne perdez plus jamais votre travail.

  • Git : init, add, commit, push, pull
  • Création et gestion de dépôts GitHub
  • Branches et pull requests
  • Collaboration en équipe
  • Gestion des conflits de code
  • README professionnel
⏱ Durée : 3 jours — Niveau : Débutant
06
🚀
Déploiement en ligne

Publiez votre site sur GitHub Pages ou Firebase Hosting gratuitement.

  • Déploiement sur GitHub Pages
  • Firebase Hosting (domaine personnalisé)
  • Nom de domaine personnalisé
  • Certificat SSL (HTTPS)
  • Optimisation des performances
  • Projet final : site en ligne complet
⏱ Durée : 3 jours — Niveau : Tous niveaux
